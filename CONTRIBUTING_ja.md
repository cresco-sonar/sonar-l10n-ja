# コントリビューションガイド

[English](https://github.com/cresco-sonar/sonar-l10n-ja/blob/master/CONTRIBUTING.md)

**SonarQube 日本語翻訳パック** に関心をお持ちいただきありがとうございます。

現在、このプロジェクトでは **Pull Request による直接の翻訳修正やコード変更は受け付けておらず**、  
主に **Issue の報告** をお願いしています。

## Issue の報告方法

報告をしやすくするために、専用の Issue テンプレートを用意しています。

1. このリポジトリの **Issues** タブを開きます。
2. **New issue** ボタンをクリックします。
3. テンプレート一覧から **「翻訳の問題報告」** を選択します。
4. 項目をすべて埋めてください：
   - **概要**：問題の簡単な説明
   - **詳細**：
     - 画面 / ページ：問題が発生した場所
     - 現在のテキスト：今表示されている文章
     - 期待されるテキスト：正しい / 自然な文章
   - **再現手順**：
     1. ステップ1
     2. ステップ2
     3. ステップ3
   - **その他の情報**：スクリーンショットや参考資料（任意）

### 報告例
- UI 内の日本語が不自然
- 特定メッセージの翻訳が抜けている
- 誤字や文法の誤り

## ライセンスについて

Issue に記載された提案や改善案は、このプロジェクトに組み込まれ、  
**GNU LGPL v3** ライセンスのもとで配布されることに同意いただいたものとします。
