# Contributing Guide

[日本語](https://github.com/cresco-sonar/sonar-l10n-ja/blob/master/CONTRIBUTING_ja.md)

Thank you for your interest in improving the **Japanese translation pack for SonarQube**!

At this time, we are mainly accepting **Issue reports** rather than direct code or translation contributions via Pull Requests.

## How to report an Issue

We have prepared an Issue template to make reporting easier.

1. Go to the **Issues** tab of this repository.
2. Click **New issue**.
3. Select **"Translation Issue"** from the list of templates.
4. Fill in all required sections:
   - **Summary**: short description of the problem
   - **Details**:
     - Screen / Page: where the problem occurs
     - Current Text: what is displayed now
     - Expected Text: what should be displayed
   - **Steps to reproduce**:
     1. Step one
     2. Step two
     3. Step three
   - **Additional info**: screenshots or references (optional)

### Examples
- Incorrect or unnatural Japanese translation in the UI
- Missing translation for a particular message
- Typo or grammatical error
